### Naming Convention
For **Pages** use camel case
```
systemSettings
```

For **Components** use kebab case
```
sup-admin-panel
```

For **Router** use kebab case
```
system-settings
```

### Files
Vue files use Pascal Case
```
SupAdmin.vue
```

Typescript files use Pascal case
```
AuthenticateUser.ts
```
